Privacy Policy

This GPT Action is designed to retrieve bibliometric data (e.g., Journal Impact Factor) based on journal names provided by the user. The following outlines our data practices:

Data Handling

  • Journal Name Transmission: When a user provides a journal name, it is sent to a secure backend server hosted on Render.com, located in Frankfurt, Germany (European data center).
  • Data Retrieval: The server queries an external API to retrieve Journal Citation Report (JCR) data associated with the given journal name, including metrics such as Journal Impact Factor (JIF) and 5-Year JIF.

Personal Data

This GPT Action does not collect, store, or process any personal or identifiable information from users.

Data Sharing

No data is shared with third parties beyond the journal name being transmitted to the JCR API provider for information retrieval. No information is stored or reused across sessions.
